# MetaCoin Truffle Example

![Solidity](https://img.shields.io/badge/Solidity-Smart%20Contracts-blue)
![Truffle](https://img.shields.io/badge/Framework-Truffle-orange)
![Node.js](https://img.shields.io/badge/Runtime-Node.js-green)
![License](https://img.shields.io/badge/License-Educational-lightgrey)

This repository contains a **sample cryptocurrency smart contract project** built using **Solidity and the Truffle Framework**.

The project demonstrates how to **create, compile, test, and deploy a simple cryptocurrency** on an Ethereum development blockchain.

---

# 📚 Full Tutorial

A complete step-by-step tutorial is available here:

👉 https://www.blockchainguide.biz/crypto-tutorial/index.html

The tutorial explains how to:

- Install the blockchain development environment
- Set up Node.js and Truffle
- Create a MetaCoin smart contract
- Compile and test smart contracts
- Interact with smart contracts using Web3
- Deploy the contract using Ganache

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/liewvk/metacoin-truffle-example.git
```

Enter the project directory

```bash
cd metacoin-truffle-example
```

Install project dependencies

```bash
npm install
```

---

# 🚀 Compile the Smart Contract

Compile the smart contracts using Truffle:

```bash
truffle compile
```

---

# 🧪 Run Tests

Run the smart contract tests:

```bash
truffle test
```

---

# 🔗 Deploy to Local Blockchain

Start the Truffle development console:

```bash
truffle develop
```

Deploy the contract:

```bash
migrate
```

---

# 🌐 Deploy Using Ganache

If you are using **Ganache**, run:

```bash
truffle migrate --network ganache
```

Ganache provides a local Ethereum blockchain environment for testing smart contracts.
