# MetaCoin Truffle Example

This repository contains a **sample cryptocurrency smart contract project** built using the **Truffle Framework and Solidity**.

The project demonstrates how to create, compile, test, and deploy a simple cryptocurrency on an Ethereum development blockchain.

---

## 📚 Full Tutorial

A complete step-by-step tutorial is available here:

👉 https://www.blockchainguide.biz/crypto-tutorial/index.html

The tutorial covers:

- Installing the blockchain development environment
- Setting up Truffle and Node.js
- Creating a MetaCoin smart contract
- Compiling and testing smart contracts
- Interacting with the contract using Web3
- Deploying the contract to Ganache

---

## 🧱 Technologies Used

- Solidity
- Truffle Framework
- Web3.js

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/liewvk/metacoin-truffle-example.git
````

Enter the project directory:

```bash
cd metacoin-truffle-example
```

Install dependencies:

```bash
npm install
```

---

## 🚀 Compile the Smart Contract

```bash
truffle compile
```

---

## 🧪 Run Tests

```bash
truffle test
```

---

## 🔗 Deploy to Local Blockchain

Start the Truffle development console:

```bash
truffle develop
```

Then deploy the contract:

```bash
migrate
```
